# Grocery Billing System

## Problem Statement
A grocery store wants to calculate the total cost of 3 items.
If the total exceeds $50, a 10% discount is applied.
The program displays the original total, discount, and final payable amount.

## Approach
- Accept input for 3 items
- Calculate total cost
- Apply 10% discount if total > 50
- Display results

## Sample Output

Enter price of item 1: 25
Enter price of item 2: 20
Enter price of item 3: 15

Original Total: $ 60.0
Discount: $ 6.0
Final Payable Amount: $ 54.0
